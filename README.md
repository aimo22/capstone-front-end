# Capstone-Front-End

## Description
The front-end is built with React, connecting to a Python Flask backend. It fetches data from the Rescue Group API for pet adoption, and uses mongoDB with backend routes enabling user profiles and favorites. [Click here for the backend.](https://github.com/aimo22/capstone-full-back-end)

## List of dependencies
- React
- React Router
- Axios

## Instructions for setting up the App

1. Clone this repository locally:
   ```
   $ git clone [repository_url]
   ```

2. Open the project directory:
   ```
   $ cd capstone-front-end
   ```

3. Start the React app:
   ```
   $ yarn start
   ```
